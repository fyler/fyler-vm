## About

VM configuration for [fyler](https://github.com/palkan/fyler) development

Installs:

* required system packages
* erlang 
* aws client
* postgres
* unoconv
* swftools
* ffmpeg (with aac, vpx, h264, speex, opus support)
* imagemagick
* pdftk
* ...

## Requirements

* Vagrant >=1.4
* VirtualBox >=4.0
* [Ansible](http://docs.ansible.com/intro_installation.html) >=1.7
